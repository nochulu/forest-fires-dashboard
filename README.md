NDVI (Normalized Difference Vegetation Index) — это стандартизированный вегетационный индекс для оценки состояния растительности. Он показывает количество активной зеленой массы и часто используется в точном земледелии для мониторинга всходов, контроля болезней и прогнозирования урожая.
EVI (Улучшенный вегетационный индекс — Enhanced Vegetation Index)Назначение: Оценка плотности и здоровья зеленой растительности.
NBR (Нормализованный индекс выгорания — Normalized Burn Ratio)Назначение: Оценка наличия влаги, выявления гарей и мониторинг лесных пожаров.
